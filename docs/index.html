PaperCoat

Turn any novel PDF into a convincing academic paper.
Read fiction at work without raising eyebrows.

Stay productive. Read fiction.

⬇ Install 📖 Docs GitHub
$ curl -fsSL https://papercoat.dev/install.sh | sh

✦ Features

⚡

Single Binary

2.5 MB, zero dependencies. Download and run — no Python, no npm, no Docker.

🖥️

Interactive TUI

Full terminal UI with file browser, live preview, and keyboard-driven controls.

🎭

Deep Disguise

Fake DOIs, volume/issue numbers, journal metadata, ORCIDs, and review dates.

📚

Journal Templates

Default (two-column), Nature, and IEEE formats. Each with proper typography.

📊

Text Analytics

Word count, readability scores, sentence stats — turn any text into data.

📂

Batch Mode

Process entire directories of PDFs with a single command.

✦ How It Works

1

Extract

Text is extracted from any PDF using high-performance Rust libraries.

2

Transform

Raw text is structured into a paper: title, authors, abstract, sections, references.

3

Disguise

Realistic journal metadata, DOIs, and citations are generated automatically.

4

Render

Output is generated as compilable LaTeX or compiled PDF with full metadata.

✦ See It In Action

papercoat
$ papercoat ~/Downloads/novel.pdf
